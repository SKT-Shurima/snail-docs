    # Vim

    • 按ESC跳到命令模式

      命令 说明
      :w 保存文件，不退出 vim
      :w file 将修改另存到 file 中，不退出vim
      :w! 强制保存，不退出vim
      :wq 保存文件，退出vim
      :q 不保存文件，退出vim
      :q! 不保存文件，强制退出
      :e! 放弃所有修改，从上次保存文件开始再编辑
