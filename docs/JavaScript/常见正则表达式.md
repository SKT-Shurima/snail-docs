###  常见正则表达式

| 类型           | 正则                                |
| -------------- | :---------------------------------- |
| 判断是否是中文 | `/[\u4e00-\u9fa5]/`                 |
| 邮箱           | `/^(\w-*\.*)+@(\w-?)+(\.\w{2,})+$/` |
| 数字           | `/^[0-9]+.?[0-9]*$/`                |
| 手机号         | `/^1[3456789]\d{9}$/`               |
|                |                                     |
|                |                                     |
|                |                                     |
