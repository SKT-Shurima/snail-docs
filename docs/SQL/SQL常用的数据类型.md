### 常用数据类型

| 数据类型     | 说明                              |
| ------------ | --------------------------------- |
| int          | 整数类型                          |
| char(n)      | 字符/字符串 固定长度              |
| varchar(n)   | 字符/字符串 可变长度，最大长度为n |
| numeric(a,b) | 精确数值，总位数a，小数点后位数b  |
| date         | 存储年月日的值                    |
|              |                                   |
